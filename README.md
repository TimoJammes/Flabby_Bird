# Flabby Bird

A Flappy Bird-inspired game built in Python with Pygame, developed during the **CentraleSupélec 2024 Coding Week** in Nantes.

![Team Logo](Images/Logo.jpg)

---

## Features

- Configurable background, bird color, and difficulty
- Moving pipes mode
- Shooter mode (dodge bullets as well as pipes)
- In-game settings menu with keyboard navigation

---

## Prerequisites

- Python 3.9–3.13 (Pygame does not yet support 3.14)
- [Pygame](https://www.pygame.org/)

---

## Getting Started

```bash
# Clone the repository
git clone git@gitlab-student.centralesupelec.fr:rita.guerraoui/flappy_bird_by_one_team.git
cd flabby_bird

# Install dependencies (use pip3 / python3 on macOS) (Creating a virtual environment is recommended)
pip install pygame

# Run
python main.py
```

---

## How to Play

| Action | Key |
|---|---|
| Flap | `Space` |
| Navigate menu | `↑` / `↓` |
| Change setting value | `←` / `→` |
| Replay after death | `Space` |
| Return to main menu | `Backspace` |

Avoid pipes, the ground, and bullets (in shooter mode). Score one point per pipe cleared.
