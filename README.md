## 🐦 **Project Overview: Flappy Bird**  

### 📜 **Description**  

Vietnamese programmer and video game artist **Dong Nguyen** created the casual mobile game **Flappy Bird** in 2013 through his game production company `GEARS`.  

In this side-scrolling game, the player controls *Faby*, a bird that must navigate through green pipe columns without colliding with them.  

The player's score is determined by the number of pipes successfully passed.  

We based ourselves on this game for our MVP, and added many new and unique features to it, creating Fla**bb**y Bird.

# 🖥 Terminal Execution Guide to Run the Game 🔧

Follow these simple steps to get started:


## 1️⃣ Open the Terminal Application  
Launch the Terminal on your device to begin.


## 2️⃣ Check python version
Ensure you have the latest version of Python (3.13 as of 11/21/2024) by running the following command, and install it if necessary at https://www.python.org/downloads/:

(WARNING: pygame is not supported yet by Python 3.14 as of 4/27/2026)
**Windows:** 
```bash
python --version
```
**Mac:**  
```bash
python3 --version
```

## 3️⃣ Clone the Repository  
Navigate to the folder you would like to clone the repository and run the following command to clone the repository:  

```bash
git clone git@gitlab-student.centralesupelec.fr:rita.guerraoui/flappy_bird_by_one_team.git
```

## 4️⃣ Navigate to the repository in your files
Run the following command to navigate to the correct folder:

```bash
cd flabby_bird
```

## 5️⃣Install Pygame

Ensure the python module **Pygame** is installed on your computer by typing the appropriate following command in your Terminal:


**Windows:** 
```bash
pip install pygame
```
**Mac:**  
```bash
pip3 install pygame
```

## 6️⃣ Run the Game in your terminal

   **Windows:**

```bash
python main.py
```

**Mac:** 

```bash
python3 main.py
```


## 🎮 **How to Play Flabby Bird**  

**Menu:**
- In the **Main Menu**, navigate through the settings (background image, bird color, game difficulty, *moving tubes* game mode, *shooter* game mode) using the *up and down arrow* keys, and change their values using the *left and right arrow* keys.

**Game:**
1. **Fly**: Press the spacebar to make the bird flap its wings and gain altitude.  
2. **Avoid Obstacles**: Navigate through gaps between pipes. Hitting a pipe or the ground (or a bullet in the *shooter* game mode!) ends the game.  
3. **Score Points**: Earn points for every gap successfully passed.  
4. **Keep Flying**: Maintain a steady rhythm to keep the bird airborne while avoiding collisions.  

**Pro Tip**: Timing is key! 

**Death Menu:**

- After dying, in the **Death Menu**, either press the *Space* key to play again, or the *Backspace* key to return to the main menu.
